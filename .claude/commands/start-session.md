Starting a new session. Follow these steps:

## 1. Load Project Context

Read these files to restore context:
1. **CLAUDE.md** - Full project context
2. **PROJECT-TODO.md** - Current tasks and status
3. **CHANGELOG.md** - Recent history (last entry)

## 2. Summarize

Provide a brief summary:
- Project purpose and current phase
- Last session's accomplishments
- Immediate priorities (next unchecked items in TODO)
- Any blockers or pending questions

## 3. Ready to Work

Ask the user what they want to focus on today.
