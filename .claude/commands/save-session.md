Save session state. Follow these steps:

## 1. Update CHANGELOG.md

Add a new dated entry with:
- **Accomplished:** What was done this session
- **Decisions:** Any key decisions made
- **Added/Changed:** Files created or modified
- **Next Steps:** Priorities for next session

## 2. Update PROJECT-TODO.md

- Mark completed items with `[x]`
- Mark in-progress items with `[~]`
- Add any new tasks discovered
- Update blocked items with `[?]` and reason
- Update the "Last Updated" date

## 3. Update Technical Docs

If technical work was done, update relevant files in `technical/`:
- `architecture.md` for design changes
- `api-reference.md` for new endpoints
- `deployment.md` for infrastructure changes

## 4. Summary

Display:
- Number of tasks completed this session
- Number of new tasks added
- Current blocked items (if any)
- Top 3 priorities for next session
